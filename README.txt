NutriVida - upgrade busca e ficha do alimento

Como usar:
1. Substitua os arquivos antigos por estes arquivos.
2. Mantenha o seu taco.json na mesma pasta dos HTMLs.
3. Abra index.html.

Novidades:
- Busca rápida dentro do modal "Adicionar alimento".
- Lista de sugestões enquanto digita.
- Clique no alimento na aba Alimentos para abrir ficha completa.
- Ficha com nutrientes por 100g e gráfico colorido de macros.
- Botão direto para adicionar o alimento ao diário pela ficha.

Arquivos principais:
- index.html
- alimentos.html
- app.js
- app.css
