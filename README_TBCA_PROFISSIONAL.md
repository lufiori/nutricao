# NutriVida TBCA Profissional

Esta versão usa `tbca_pronto.json` na página `alimentos.html`.

## Como gerar o tbca_pronto.json

1. Coloque o arquivo bruto `tbca.json` nesta pasta.
2. Rode:

```bat
py converter_tbca.py
```

Isso gera:

```text
tbca_pronto.json
```

## Subir no GitHub

Suba juntos:

- alimentos.html
- app.css
- app.js
- converter_tbca.py (opcional, só para gerar localmente)
- tbca_pronto.json
- demais arquivos do projeto
